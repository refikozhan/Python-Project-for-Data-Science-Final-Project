Summary
The purpose of this tutorial is to teach basic Python skills for working with data. Successful completion of this course includes practice on an application project that you will develop using Python. This project includes an approach that transforms data into an interactive dashboard.


Project Topics
*Analyzing Historical Stock/Revenue Data and Building a Dashboard
* Extracting Stock Data Using a Python Library
* Extracting Stock Data Using Web Scraping


Learned libraries
* Pandas
* BeautifulSoup
* yfinance
* Matplotlib
* Scipy
* html5lib
* Plotly
* requests
